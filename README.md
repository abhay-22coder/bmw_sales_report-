# 🚗 BMW Global Sales Dashboard (2018–2025)

## 📌 Project Overview
This is my first Power BI project, where I analyzed BMW global sales data from 2018 to 2025. The goal of this dashboard is to track sales performance, revenue trends, and business growth through interactive visualizations.

The project helped me practice data cleaning, data modeling, DAX calculations, and dashboard design in Power BI.

---

## 🎯 Objectives
- Analyze BMW sales performance over time.
- Monitor revenue trends and growth.
- Compare yearly sales results.
- Identify top-performing periods.
- Create an interactive dashboard for business decision-making.

---

## 🛠 Tools Used
- **Power BI**
- **DAX (Data Analysis Expressions)**
- **Power Query**
- **Excel/CSV Data Sources**

---

## 📊 Dashboard Features
- Total Revenue KPI
- Total Sales KPI
- Year-over-Year (YoY) Analysis
- Previous Year Sales Comparison
- Revenue Trend Analysis
- Interactive Filters and Slicers
- Dynamic Visualizations

---

## 📈 Key Insights
- Sales performance trends from 2018–2025.
- Revenue growth across different years.
- Identification of peak sales periods.
- Comparison between current and previous year performance.
